# 🏦 Sistema de Gestión de Cuenta Bancaria (POO en Python)

Este es un proyecto práctico desarrollado en Python que simula las operaciones básicas de una cuenta bancaria utilizando Programación Orientada a Objetos (POO). 

El script define una clase `CuentaBancaria` que permite gestionar el saldo de un usuario mediante operaciones seguras como depósitos y retiros, incluyendo validaciones lógicas para evitar 
transacciones con fondos insuficientes.

## 🚀 Características

* **Gestión de Cuentas:** Inicializa objetos con el nombre del titular, número de cuenta y saldo inicial.
* **Depósitos:** Permite incrementar el saldo de la cuenta mostrando un mensaje de confirmación de la transacción.
* **Retiros Seguros:** Implementa validaciones lógicas (`if/else`) para asegurar que el usuario no pueda retirar más dinero del que tiene disponible, devolviendo un mensaje de error si hay "Saldo insuficiente".
* **Consulta de Saldo:** Métodos dedicados para retornar el saldo actual y formatear la información del usuario de manera clara.

## 🧠 Conceptos de Programación Aplicados

Este repositorio es una excelente demostración de los siguientes conceptos:
* **Clases y Objetos:** Modelado de entidades del mundo real (una cuenta de banco) en código.
* **Encapsulamiento y Estado:** Modificación del estado interno del objeto (`self.saldo`) a través de métodos específicos.
* **Estructuras de Control:** Uso de condicionales para proteger la integridad de los datos.
* **F-Strings:** Formateo dinámico de cadenas de texto para mostrar información personalizada del usuario.
