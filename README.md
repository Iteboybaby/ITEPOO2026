# 🐍 Ejercicios de Programación Orientada a Objetos (POO) en Python

Este repositorio contiene una serie de ejercicios prácticos desarrollados en Python, enfocados en aplicar los pilares de la Programación Orientada a Objetos: Encapsulamiento, Abstracción, Herencia y Polimorfismo.

## 📂 Estructura del Proyecto

El repositorio está dividido en diferentes directorios, cada uno enfocado en un escenario específico de modelado de clases:

* **`Ejercicio 1 Estudiante`**: 
    Implementación de clases básicas, constructores y definición de atributos y métodos para gestionar información académica.
* **`Ejercicio 2 bancaria y mascota`**: 
    Modelado de objetos del mundo real. Prácticas de encapsulamiento para proteger datos sensibles (como el saldo de una cuenta) y gestión del estado de los objetos.
* **`Ejercicio 3 animales`**: 
    Aplicación práctica de la **Herencia**. Creación de clases padre e hijas para reutilizar código y entender el polimorfismo a través del comportamiento de diferentes animales.
* **`Ejercicio 4 Aventurero`**: 
    Lógica de programación enfocada en videojuegos. Atributos de personajes, inventarios o interacciones de combate usando objetos.
* **`Ejercicio 4 Restaurante`**: 
    Sistema de gestión básico. Interacción entre múltiples clases (como pedidos, mesas, clientes y menú).
* **`Ejercicio 5 Abstraccion`**: 
    Uso avanzado de clases abstractas para definir plantillas estrictas que otras clases deben seguir, asegurando una arquitectura de software sólida.
